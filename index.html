Personal Projects

EV Ownership Analyzer

A WinUI 3 application that takes data from the Wallbox level 2 charger, Entergy, gas price data from the US eia API, and the KIA Owners Portal to calculate the cost of ownership of and EV compared to ICE vehicle.

EV Ownership Analyzer
La Crosse Weather Data Analyzer

This WinUI3 application was made to parse weather data exported from the La Crosse View mobile app on either Android or iOS Once loaded, there are readouts for highest and lowest outdoor, feels like temperature, and barometric pressure; highest day and total rain amount and highest wind speed.

La Crosse Weather Data Analyzer
LEGO Rebrickable Set Inventory

This WinUI3 application was made to open exported parts lists from Rebrickable and provide a local database of parts owned, still needed and other statistical information about the parts for the custom LEGO set.

LEGO Rebrickable Set Inventory
Plex Database Parser

This WinUI3 application was created to replace the manual Microsoft Excel spreadsheet being used to inventory movies, holiday specials, TV shows, etc. on the Plex MyCloud Pro Series PR2100. The application can parse the available libraries on the Plex server and then export the results to Excel to be shared.
